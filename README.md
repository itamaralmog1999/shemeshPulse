# shemeshPulse ☀️📈

shemeshPulse is a machine learning-based framework designed to forecast solar irradiance (GHI) and power output for solar energy companies. By predicting solar production, the project helps operators optimize grid integration, manage battery storage, and prevent imbalance penalties.

## 🚀 Project Objectives
* **Predictive Analytics:** Forecast Solar Irradiance (GHI) for short-term (Intraday) and mid-term (Day-Ahead) horizons.
* **Feature Engineering:** Utilize solar geometry metrics (Zenith, Azimuth) via `pvlib`.
* **Optimization:** Support commercial solar farms in trading and storage scheduling.

## 🛠️ Tech Stack (Planned)
* **Language:** Python
* **Data Science:** Pandas, NumPy, Scikit-Learn
* **Solar Physics:** pvlib-python
* **Machine Learning:** XGBoost / LSTM (Time-Series)
